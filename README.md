# US-States-Guesser

You may install the packages used in this repo via following steps:
1. python3 -m venv venv (#Creates a virtual environment) (Replace 'python3' with 'python' based on your global python version)
2. source venv/bin/activate (#Activates the virtual environment)
3. pip install -r requirements.txt

If you wish to run the game in your global python, you may simply follow the third command which will install the packages directly in your global python.
